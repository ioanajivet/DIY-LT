MongoDB Colections

A. For running the widget

1. user_data = data for each user
    course_branch_id: String
    hashed_user_id: String
    standard: String - 4 options: "completer", "explorer", "certificate", "default"
    metric_values: Array[Double] - 12
    metric_scaled_values: Array[double] - 12

2. standard_data = data for the standards used - contains 4 x (nr_weeks) docs
    course_branch_id: String
    course_week: Integer
    standard: String - 4 options: "completer", "explorer", "certificate", "default"
    metric_values: Array[Double] - 12
    metric_scaled_values: Array[double] - 12

3. widget_configuration
    course_branch_id: String
    course_week: Integer - 3 options: 1,2,3 (each 2 weeks)
    hashed_user_id: String
    selected_metrics: Array[Boolean] - 12


B. For research

- daily values for each learner = daily screenshot of user_data
- standard_data table
- weekly screenshot of widget_configuration


====

- order of indicators on the widget are randomised
- at the beginning of each week, learners:
    - can change their goal => adjusts the standards used on the widget
    - can/must selected indicators for feedback => adjusts the metrics displayed on the widget
